One of my favorite time periods to study is WW2 because of it's grand scope and lessons for humanity. Thus, I searched Kaggle.com for WW2 datasets. I stumbled across a comprehensive dataset containing information about the German Kriegsmarine (Navy). The dataset contains records on Uboat (submarine) production and it's missions.

These boats are infamous for making up the bulk of the German Navy and are a crucial part of naval history. During WW1 and WW2, the Uboats were quite threatening to allied forces which changed how they approached the war. By conducting an analysis of this dataset, I am excited to find patterns and see how much of an impact the Uboats had. I am also curious to see if the data will differ as the war raged on.

Ultimately, the goal of this project is to nerd out with historical data!

Exploratory Data Analysis




Despite the dataset recapping data from 80+ years ago, it is packed with data. I plan to utilize the ships sunk data to find the average and median for the boats. I can take this analysis a step further by grouping the table by type. This will allow me to analyze each type individually. Furthermore, I would like to find the spread for the ships sunk data and determine the most succsesful Uboats. By finding the spread, I could potentially see if a certain type of uboat performed better or worse on average. Finally, I want to attempt to perform an analysis with the "Notable Commanders" column. I can use a groupby statement to see which commanders sunk the most amount of tonnage on average

Note: In the context of naval technology, tonnage is a term used to define the size of a ship. If two navys clash, the side that sinks the most amount of tonnage will strictly win the battle.

def uboatPie(n=10):
    
    data = []
    for i in range(n):
        trash_boat = random.randint(0, len(df) - 1)
        dic = {"Fate Event": df['Fate_Event'].iloc[trash_boat], "Name": df['Name'].iloc[trash_boat], "Date": df['Fate_Date'].iloc[trash_boat]}
        data.append(dic)
    sample_fates_df = pd.DataFrame(data)

    return sample_fates_df
